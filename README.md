## What is it?

1. Deploy ML Models

   * Building and automating ML pipelines
     * MLOps
     * Airflow DAG
     * Cloud Services(AWS/GCP/Azure)

     * Streamlit
     * RESTAPI
     * Docker

      * Residual Analysis
      * Model Remediation
      * Model Fairness
